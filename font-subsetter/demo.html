Ako Použiť

Stačí nakopírovať výstupné súbory do vášho projektu:

output/ ├── sans-serif/ │ ├── roboto/ → Roboto.{ttf,woff,woff2,css} │ └── ... └── serif/ ├── cormorantgaramond/ → CormorantGaramond.{ttf,woff,woff2,css} └── ...

V HTML hlavičke:

<link rel="stylesheet" href="fonts/sans-serif/roboto/Roboto.css"> <link rel="stylesheet" href="fonts/serif/playfairdisplay/PlayfairDisplay.css">

V CSS:

body { font-family: 'Roboto', sans-serif; } h1, h2 { font-family: 'Playfair Display', serif; }
